# SweetCraft - Retail & Sweet Shop Manager 🍬

A modern, responsive web application tailored for sweet shops and retail stores to manage inventory, staff payroll, party orders, and customer debts.

## 🚀 Features
* **Inventory & Expiry Alerts:** Track stock and get automatic alerts for expiring items.
* **Party Orders Kanban:** Manage bulk orders with visual status tracking and due amount calculations.
* **Advanced Staff Khata:** Handle daily wages, monthly salaries, advances, and single-click attendance.
* **Customer Udhari Book:** Keep a complete ledger of customer debts and payments.

## 🛠️ Tech Stack
* **Frontend:** React.js, Vite, Tailwind CSS, Lucide Icons
* **Backend:** Python, Flask, SQLAlchemy
* **Database:** SQLite
* **Deployment:** Docker & Docker Compose

## 🐳 Quick Start (Docker)
1. Clone the repository: `git clone https://github.com/rajatpoddar/SweetCraft.git`
2. Run `docker-compose up -d --build`
3. Access the app at `http://localhost:8080`
